## 💻 Getting Started <a name="getting-started"></a>


To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need:



```sh
  nodejs
  your favored web browser 
  Vscode Optional
```
 
### Setup

Clone this repository to your desired folder:
Server Repo: https://github.com/belaymit/hello-rails-back-end/tree/hello-rails-back-end
- clone this repo
``` sh
  git checkout hello-rails-back-end
```


```sh
  cd my-folder
  git clone ggit@github.com:belaymit/hello-react-front-end.git
```


### Install

Install this project with:


```sh
  cd hello-react-front-end
  npm install
```


### Usage

To run the project, execute the following command:


```sh
  Run the server using rails s then
  npm start
```
**Note** the react front-end app and rails back-end must run in different port

The sever Rails app at port localhost:3000 The react app ``localhost:3001`` or other than ``Port: 3000 or ````localhost == 127.0.0.1``
